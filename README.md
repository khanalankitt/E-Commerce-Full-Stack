# E-Commerce Website

A modern full-stack e-commerce website built with **Next.js** for the frontend and **Express.js** for the backend.

## Tech Stack

### Frontend

* Next.js
* TypeScript
* Tailwind CSS

### Backend

* Express.js
* TypeScript
* MongoDB

## Status

🚧 **This project is currently under development.**

New features, improvements, and documentation will be added as development progresses.
